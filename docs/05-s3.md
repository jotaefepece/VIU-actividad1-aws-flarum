Desde la consola de AWS se accedió al servicio **Amazon S3**, el cual permite almacenar archivos en la nube. Este servicio será utilizado para crear el bucket solicitado en la actividad y almacenar en él un documento PDF y una imagen JPG con acceso público.

## 5. Inicio de la creación del bucket S3

Dentro del servicio Amazon S3, se seleccionó la opción **Create bucket** para comenzar la configuración del nuevo bucket requerido por la actividad.

## 6. Configuración del nombre y la región del bucket

Se configuró el bucket con el nombre **hfjpec-actividad1-08masw**, siguiendo el formato solicitado en la actividad a partir de las iniciales de los integrantes del grupo. Además, se seleccionó la región **us-east-1 (N. Virginia)**, tal como exige el enunciado.

## 7. Configuración de acceso público del bucket

Se deshabilitó el bloqueo de acceso público del bucket S3, con el objetivo de permitir la descarga de archivos desde internet. Esta configuración es necesaria para cumplir con la actividad, ya que se requiere generar URLs públicas para un documento PDF y una imagen JPG almacenados en el bucket.

Se desactiva el bloqueo de acceso público, lo cual es necesario para permitir el acceso mediante políticas de bucket.

## 8. Configuración de cifrado del bucket

Se configuró el cifrado del bucket utilizando Server-Side Encryption con claves administradas por Amazon S3 (SSE-S3). Esto permite proteger los datos almacenados en el bucket sin necesidad de gestionar claves manualmente.

## 9. Creación del bucket S3

Se creó el bucket S3 con el nombre hfjpec-actividad1-08masw en la región us-east-1 (N. Virginia), con acceso público habilitado y cifrado SSE-S3 configurado, cumpliendo los requisitos establecidos en la actividad.

## 10. Inicio de la carga de archivos al bucket

## 13. Verificación de objetos almacenados

## 15. Configuración de acceso público mediante política de bucket

Debido a que las listas de control de acceso (ACL) se encuentran deshabilitadas en el bucket, se configuró una política de bucket para permitir el acceso público a los objetos almacenados. Se definió una política que autoriza a cualquier usuario a realizar operaciones de lectura (GetObject) sobre los archivos del bucket.

Esta política permite acceso de lectura pública a todos los objetos del bucket.

**Política aplicada**

{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "PublicReadGetObject",
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::hfjpec-actividad1-08masw/*"
    }
  ]
}
