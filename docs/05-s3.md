## 1. Activación del laboratorio en AWS Academy

Se inició el laboratorio desde AWS Academy mediante la opción **Start Lab**. Una vez cargado el entorno, se verificó que el icono de AWS apareciera en color verde, lo que confirma que la sesión está activa y que ya es posible acceder a la consola de administración de AWS.

## 2. Acceso a la consola web de AWS

Con el laboratorio ya activo, se accedió a la consola web de AWS haciendo clic sobre el icono de AWS habilitado en AWS Academy. Esta consola permite administrar los servicios necesarios para la actividad, entre ellos Amazon S3, EC2 y RDS.

## 3. Verificación de la región de trabajo

Se verificó que la región seleccionada en la consola de AWS fuera **us-east-1 (N. Virginia)**, ya que la actividad especifica que el bucket S3 debe crearse en esa región.

## 4. Acceso al servicio Amazon S3

Desde la consola de AWS se accedió al servicio **Amazon S3**, el cual permite almacenar archivos en la nube. Este servicio será utilizado para crear el bucket solicitado en la actividad y almacenar en él un documento PDF y una imagen JPG con acceso público.

## 5. Inicio de la creación del bucket S3

Dentro del servicio Amazon S3, se seleccionó la opción **Create bucket** para comenzar la configuración del nuevo bucket requerido por la actividad.
